Climaproof Model Selection Tool

The Model Selection Tool compares climate change signals of different EURO-CORDEX Models used in the ClimaProof project. The climate change signals are calculated with reference to the period 1981-2010.

Interact with the widgets on top to create the visualization. Hover over the circles to see more information about the specific model.

Download the data via data.ccca.ac.at
